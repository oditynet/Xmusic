# Xmusic
Для чата eXpress.ms добавил возможность загрузки музыки через микрофон. Ранее чат не поддерживает проигрывание аудифайлов встроенным плеером. Приходилось скачивать mp3 и слушать

https://addons.mozilla.org/ru/firefox/addon/xmusic/

Ожидаем обобрение АМО

#### Описание

Ух, ну и задачка..... Кайфанем!?
Файлы console - копируете весь текст . В соответствующем браузере на вкладке express чата нажимаете F12 ->console. Вставляете код из файла . Нажимаете enter. Выбираете файл музыкальный. Дальше скрипт его через микрофон запишет,сам файл отправит в чат. Profit. 
Если вы счастливый обладатель браузера firefox то вот вам аддон:
Firefox->menu-> расширение и темы->Шестеренка-> отладка дополнений->загрузить временное дополение. Выбираете zip архив. Обновляете чат. снизу будет кнопка. загружаете музон. Профит.
1) Да, запись идет через микрофон. ждать придется долго. Но вкладки можно в браузере менять, а не сидеть и ждать загрузки( в теории )
2) Иногда адон файл не отправляет в чат. надо увеличить таймаут - руками нажмете кнопку.
3) Понравится - могу в addons.firefox.com добавить этот аддон чтоб было все официално и красиво.

#### Фиаско

Оказывается express подедрживает через API загрузку аудиофайлов и их прослушивание встроенным плеером. Ждем релиза, а пока можно пощупать так. Открываем файд mp3.txt. В консоли F12 выполняем пункт 1 и потом выполняем пункт 2. Отправляем сообщение и слушаем
